Connect to Google Sheets
All data syncs to your own Google Sheet. Cloud backup. Accessible from any device.
STEP 1 — Create your Google Sheet
Go to sheets.google.com, create a new sheet named Master Financial Control. Copy the ID from the URL.
https://docs.google.com/spreadsheets/d/[COPY THIS PART]/edit
STEP 2 — Add the sync script (supports backup AND restore)
In your sheet: Extensions → Apps Script. Delete existing code, paste the script below, click Save, then Deploy → New Deployment → Web App. Set Execute as: Me and Who has access: Anyone. Click Deploy, authorize when prompted, and copy the Web App URL.
function doPost(e){ var ss=SpreadsheetApp.openById(e.parameter.sheetId); var d=JSON.parse(e.parameter.data); var sh=ss.getSheetByName(d.tab)||ss.insertSheet(d.tab); if(d.clear)sh.clearContents(); if(d.headers&&sh.getLastRow()===0)sh.appendRow(d.headers); if(d.rows)d.rows.forEach(function(r){sh.appendRow(r);}); return ContentService.createTextOutput("OK"); } function doGet(e){ var ss=SpreadsheetApp.openById(e.parameter.sheetId); var tab=e.parameter.tab||'Master Log'; var sh=ss.getSheetByName(tab); if(!sh)return ContentService.createTextOutput(JSON.stringify({rows:[]})).setMimeType(ContentService.MimeType.JSON); var data=sh.getDataRange().getValues(); return ContentService.createTextOutput(JSON.stringify({rows:data})).setMimeType(ContentService.MimeType.JSON); }
⚠ IMPORTANT — If you already deployed the old script
Replace the old code with the new script above, save, then go to Deploy → Manage Deployments → Edit (pencil icon) → Version: New Version → Deploy. Use the same Web App URL — no need to reconnect.
